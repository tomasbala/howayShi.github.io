常用框架工具-Matplotlib作业

题目描述

绘制一个函数
在区间[0,2]上绘制函数f(x) = sin^2(x-2)*e^(-x^2)，并加上轴标签和标题。
横轴：x
纵轴：f(x)
标题：f(x) = sin^2(x-2)*e^(-x^2)
提示：
1.使用np.linspace来生成[0,2]区间任意数量的横轴坐标。
2.使用np.sin和np.exp等函数来完成f(x)函数值的求解。
3.使用plt.plot来作图。
4.使用plt.xlabel, plt.ylabel来生成坐标轴标签，用plt.title来生成图像标题。

代码实现:

import numpy as np
import matplotlib.pyplot as plt 
%matplotlib inline

X=np.linspace(0,2,num=20,endpoint=True)
Y=(np.sin(X-2)**2)*np.exp(-(X**2))
plt.plot(X, Y, color="purple", linewidth=3.0, label="Blue", linestyle="-")
plt.xlabel("X")
plt.ylabel("f(x)")
plt.title("f(x) = sin^2(x-2)*e^(-x^2)")
plt.show()

Python编程知识-奇妙的三元组

题目描述

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。
注意：答案中不可以包含重复的三元组。
  示例：
  给定数组 -1 0 1 2 -1 -4
  满足要求的三元组集合为：
  -1 0 1
  -1 -1 2

代码实现:

n=input()
num=[int(t) for t in n.split()]
l=len(num)
a=[]
s=[]
b=0 #button
for i in range(l-1):
    for j in range(i+1,l-1):
        for k in range(j+1,l-1):
            if (num[i]+num[j]+num[k]== 0):
                    a= [num[i], num[j], num[k]]
                    a.sort()
                #   print(a)
            if (a not in s):
                s.append(a)
for e in s:
    if(b==1):
        print("\n")
        b=0
    for m in e:
        print(m,end=" ")
    b=1

常用框架工具-Numpy作业

题目描述

给定一个数值b, 从一个数组A中找到与b最近的元素，并输出该元素和其在数组中的位置。
提示：np.abs, argmin, flat
如：b = 0.5
A = [0.96, 0.10, 0.27, 0.49, 0.53, 0.60, 0.30, 0.89, 0.82, 0.28]
应输出：0.49， 3

代码实现：

import numpy as np
b=float(input())
A=[0.96, 0.10, 0.27, 0.49, 0.53, 0.60, 0.30, 0.89, 0.82, 0.28]
n=len(A)
PA=[]
for i in range(n-1):
    PA.append(abs((A[i])-b))
m=np.argmin(PA)
print(A[m],m)

Python编程知识-字符串作业

题目描述

输入一个字符串，求最大的没有重复字符的子字符串长度
比如：输入huaweicloudaigallery
输出 9 （huaweiclo或aweicloud或weiclouda）

代码实现：

string = input()  # 手动输入字符串  
def func():
    # 在此编写代码
    n=len(string)
    length=[]
    for i in range(n):
        substring=string[i] 
        for j in range(i+1,n):#第二个字母开始判断有无重复
            if string[j] in substring:  #in语句，判断string[j]在不在列表string里

            length.append(len(substring)) #检测到重复字母，就把此刻      
                break
            else:
                substring=substring+string[j] #如果没检测到重复字母的话，就补一个字母进去继续判断
    maxlength=max(length)
    print(maxlength)
